spostiamoci nella cartella che contiene il nostro progetto
prima di avviare il tutto installare i moduli tramite npm install

se necessario eseguire npm audit fix --force

una volta preparato l'ambiente avviare digitando npm start

L'applicazione si avvierà sulla porta 3000 -> recarsi a http://localhost:3000/

selezionando tra le varie pagine del sito avremo a disposizione le varie operazione a seconda del tipo di utente

l'utente non registrato potrà registrarsi e vedere i dati relativi ai prodotti ma non potrà inserirli nel carrello e quindi acquistarli

l'utente registrato potrà vendere e acquistare i prodotti

l'admin potrà aggiungere le categorie e visualizzare i dati di utenti, prodotti e categorie
